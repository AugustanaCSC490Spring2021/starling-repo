backend

Setup

  1. Go to project folder

cd backend

  1. Install Dependencies

npm run setup

  1. Create .env with following content

```` DB_HOST= DB_DATABASE= DB_USER= DB_PASSWORD=

PORT=3000 SECRET=AAA ````

Run

  1. Start Server

npm start

  1. Open in url or using an Endpoint tester (postman)

http://127.0.0.1:3000/
